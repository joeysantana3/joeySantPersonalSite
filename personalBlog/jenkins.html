Setting up a Jenkins server in Azure.

June 30, 2018 / by Joey Santana

If you are anything like I was, you have probably heard of Jenkins, but really have no idea what it is and why people use it. If this is you, you have come to the right place. I will attempt to explain what Jenkins is, why it's useful, how to create a Jenkins server in Azure and finally how to configure it. I am not expert, but this should at least get you to understand the basic of Jenkins.

Jenkins is an open source automation server. It's used to create "pipelines" or workflows that will automate a lot of the task involved with deploying software. I will explain how I use to help me with this website. However, it can be used for other things besides just web apps.

Link to Jenkins' official website.

Before we get started, I am going to make a few assumptions.

  • You have an Azure account you can use to test
  • You have at least some experience in any command line interface

You can probably get by without any experience, but a few concepts may be frustrating to understand at first.

If you're going to follow along, you will need your own Jenkins server to play around with. Luckily, Azure makes this pretty easy. If you have more than one Azure subscription, make sure you are in the right one. The easiest way is to scroll all the way down on the Azure portal main page and click Subscriptions. Click on your test or personal subscription.

I put my Jenkins server in its own resource group. You don't have to do this if you would prefer to organize your resource in a different way. If you want to do what I did, click the Resource Groups tab and then click the + Add button on top. Name your group. Once your new Resource Group is created, there may bay a big button that allows you to add a new resource. If not, click the + Add button on the top of the UI (User Interface). You should be able to search for "Jenkins" and the first hit will probably the one you want. It's simply named Jenkins, it's published by Microsoft, and it's in the Compute category.
